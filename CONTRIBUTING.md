# Contributing to Crucible

Thanks for your interest in this mobile AI dev platform.

## How to Contribute
- Open issues with device model, Android version, Termux version
- Include `health` output when reporting bugs
- Fork, branch, change, test, PR

## Commercial Use
Apache 2.0 for open source. For commercial use open an issue tagged `commercial-inquiry`.

## Respect
Built on a phone alongside running a small business. Constructive contributions only.
