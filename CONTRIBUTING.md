# Contributing to Crucible

Thanks for your interest in Crucible — a mobile AI dev platform built on Android/Termux.

## How to Contribute

- Open issues with device model, Android version, Termux version
- Include `health` output when reporting bugs
- Fork, branch, change, test, PR

## Good First Contributions
- Testing on non-Pixel Android devices
- Documentation improvements
- Alias suggestions
- Chipset performance notes

## Commercial Use
Apache 2.0 for open source. For commercial use open an issue tagged commercial-inquiry.
